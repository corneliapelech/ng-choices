ng-choices a reigns-like game

Interested in Angular and Bootstrap, I wanted to start a small project where I could test those technologies and have some fun with them. Since I've always wanted to tell stories or rather create something others could experience, I wanted the project to be a game. That's when I decided to try to develop a reigns-like game. I love the series and I wanted to see how one could set up something similar using Angular.

I know this is far from perfect and also still in development, but hopefully you still have fun with ng-choices!
